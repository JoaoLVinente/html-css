Principais formatações

Essas duas são as principais formatações: Negrito e Itálico

Para criar um termo em destaque, no HTML5 (semântico), basta que eu use a tag <STRONG>
Assim ó, DESTAQUE
E para um termo em itálico no HTML5 (semântico), basta que eu use a tag <em>
Desta maneira, ÊNFASE

Texto marcado

Para criar uma palavra marcada, no estilo de marca-texto, deve-se usar a tag <MARK>, marca-texto
E para mudar a cor do marca-texto, deve-se usar de duas formas (tem uma terceira que vai ser vista só mais pra frente)

1°: Coloque a "tag" (style) na frente da tag "mark", e em seguida "background color". Como no exemplo a seguir: Marca-texto

2°: Coloque a tag <style> na area da cabeça do site, ou seja, head e /head, colocando background-color. Fazendo isso, mudará a cor de todos os <mark>.

Texto deletado(excluído) e inserido(sublinhado)

Para criar um texto que tem sentido de excluído, que pode ser lido, mas não considerado, coloque a tag, <del>.
ex: Não gosto da palavra lixo.
E para colocar um texto com sentido de inserido recentemente (sublinhado), use a tag <ins>.
ex: Quero inserir/sublinhar a palavra Tânia

Citações

Para escrever citações ao invés de só usar as aspas, sua a tag <q>, pois assim, você vai estar dando um sentido de citação para aquela frase.
ex: Quando eu era pequeno, eu ouvia uma frase bem legal: filho de peixe, pexinho é.

Para criar uma citação completa, no caso de uma frase maior e algo mais "ajeitado", use a tag <blockquote>. Além disso, colocando o parâmetro "cite", posso citar o livro do qual a frase foi tirar, mais para mecanismos de busca do google e para a parte interna do site. Vejamos um exemplo:
ex: Segundo Nelson Mandela:

A educação é a arma mais poderosa que você pode usar para mudar o mundo.
